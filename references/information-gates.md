# Information Gates

## Global P0: Create

Ask only if not already provided:

- User task: create, review, outline, section, Word formatting.
- Research title or clinical question.
- Broad study nature: interventional, observational/RWE, or unknown.
- Existing input materials.

Do not make product approval status, region/use, data source, or detailed design global P0 unless absence blocks the immediate task.

## Global P0: Review

- Review goal.
- Review boundary: suggestions only, light edits, comments, redline.
- Source document/text.

Other fields should usually be extracted from the document first.

## Section-Level Gate

Each section has independent P0/P1/P2 requirements.

- P0: missing answer blocks reliable generation/review of this section.
- P1: infer and confirm.
- P2: can use placeholder or flag for later.

Many sections have no P0. Do not ask unnecessary questions.

Examples:

- Title page: usually no section P0; use title and placeholders.
- Background: usually no section P0; retrieve PubMed/guidelines and draft.
- Statistical sample size: if final calculation is requested, primary endpoint, design, effect assumptions, variance/event rates may be P0.
- RWE bias/confounding: data source, exposure, outcome, index date, and key confounders are often P0/P1.
